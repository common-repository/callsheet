Callsheet allows you to add actors/actresses/directors and their details to show them to end users.

== Description ==

Callsheet plugin allows you to add any actors, actresses, directors, cinematographers and writers.
You can add details of those too. Which will be displayed on frontend (in your theme).
You can add awards, education, date of birth, etc from this plugin.

All the details will be displayed on their pages.

You can add,update,delete,search records from here.

You can add their photos and showreel too.

Moreover, you can add in which films and tv they have worked and which shows are running right now.


== Installation ==

From Your Wordpress dashboard

1. Search 'Callsheet' from https://wordpress.org/plugins/
2. Download the plugin file.
3. Visit 'Plugins > Add new > upload plugin'.
4. Upload `callsheet.zip` .
5. Activate Callsheet from your Plugins page.


== Frequently Asked Questions ==

= How this plugin is helpful? =

Callsheet contains every single details of actors/actresses, which is useful to know what's his/her birthdate, or how many awards they get and in which films they have worked for how many years, etc.
